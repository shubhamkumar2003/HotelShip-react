**Github work flow **
1. fork the repo
2.clone the forked repo
3.make a new branch: git checkout -b new-branchName
4.git add .
5. git commit -m "text"
6.git push //pushing newbranch in forked repo
7.Do compare and merge from forked repo
//pushing into original repo
8. git remote add upstream "url up stream"
 or 
 git push --set-upstream upstream
//note: git remote -v :used to check the list of remote repo
9.git pull upstream main
OR 
git fetch --all 
git merge upstream/main
10.git push origin main
11.git merge origin main 
12. git push
